## How to run

You can run the code by doing:

```bash
make day[number]
```

And then :

```bash
./bin/day[number]/part[1|2].exe
```

## How to get the time

You can get the time by doing:

```bash
make timer
```

And then :

```bash
./bin/timer.exe ./bin/day[number]/part[1|2].exe
```

Note that the timer also takes the time to read the input file.
